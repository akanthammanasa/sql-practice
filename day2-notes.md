## Topics Covered
DISTINCT, Aggregate Functions, AS, GROUP BY, HAVING

## What I Learned

### DISTINCT
Removes duplicate values from results.

### Aggregate Functions
COUNT - counts rows
SUM   - adds up values
AVG   - calculates average
MAX   - finds highest value
MIN   - finds lowest value

### AS
Gives a readable nickname to a column in output.

### GROUP BY
Groups rows by a column so aggregates calculate per group.
Like a pivot table in Excel.

### HAVING
Filters groups after GROUP BY.
WHERE filters rows. HAVING filters groups.

## HAVING vs WHERE
WHERE  → filters rows    → before GROUP BY
HAVING → filters groups  → after GROUP BY
